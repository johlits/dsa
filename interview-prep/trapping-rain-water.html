Trapping Rain Water

Two Pointers Hard

Given an elevation map represented as bars of different heights, calculate how much water would be trapped after rainfall.

Approach (Two Pointers)

Water trapped at any index is determined by the smaller of the maximum height to its left and right. Use two pointers from both ends, maintaining the left and right maxima. Move the pointer at the smaller side inward, because the water level there is bounded by the smaller max.

  • Time: O(N)
  • Space: O(1)
public class Solution {
    public int Trap(int[] height) {
        int left = 0, right = height.Length - 1;
        int leftMax = 0, rightMax = 0;
        int water = 0;

        while (left < right) {
            if (height[left] < height[right]) {
                if (height[left] >= leftMax) leftMax = height[left];
                else water += leftMax - height[left];
                left++;
            } else {
                if (height[right] >= rightMax) rightMax = height[right];
                else water += rightMax - height[right];
                right--;
            }
        }
        return water;
    }
}
