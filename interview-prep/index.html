Interview Prep

Classic LeetCode problems with C# solutions

  • N-Queens Place N queens on an N×N chessboard so that no two queens attack each other.
  • Word Search II Find all words from a dictionary that can be formed in a letter grid.
  • Number of Islands Count separate islands in a binary grid.
  • Course Schedule Determine whether all courses can be completed given prerequisites.
  • Clone Graph Create a deep copy of a connected undirected graph.
  • Lowest Common Ancestor of a Binary Tree Find the lowest node that is an ancestor of two given nodes.
  • Binary Tree Maximum Path Sum Find the maximum sum along any parent-child path in a binary tree.
  • Longest Increasing Subsequence Find the length of the longest strictly increasing subsequence.
  • Coin Change Minimum number of coins needed to make a target amount.
  • Merge k Sorted Lists Merge k sorted linked lists into one sorted list efficiently.
  • Trapping Rain Water Calculate how much water can be trapped after raining on an elevation map.
  • Sliding Window Maximum Return the maximum in every contiguous window of size k.
  • Find Median from Data Stream Efficiently return the median after each number is added.
  • LRU Cache Design a fixed-capacity cache with O(1) get and put operations.
  • Word Ladder Find the shortest transformation sequence from start word to end word.
