Lowest Common Ancestor of a Binary Tree

Tree Recursion Medium

Given two nodes in a binary tree, find the lowest node that is an ancestor of both.

Approach

Recursively search left and right subtrees. If both target nodes are found in different subtrees of a node, that node is the LCA. If one target is found and the other is not, the found target itself is the LCA.

  • Time: O(N) — each node is visited once.
  • Space: O(H) for the recursion stack, where H is the height of the tree.
public class TreeNode {
    public int val;
    public TreeNode left;
    public TreeNode right;
    public TreeNode(int x) { val = x; }
}

public class Solution {
    public TreeNode LowestCommonAncestor(TreeNode root, TreeNode p, TreeNode q) {
        if (root == null || root == p || root == q) return root;

        TreeNode left = LowestCommonAncestor(root.left, p, q);
        TreeNode right = LowestCommonAncestor(root.right, p, q);

        if (left != null && right != null) return root;
        return left ?? right;
    }
}
