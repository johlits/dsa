* N-Queens: Place N queens on an N×N chessboard so that no two queens attack each other. Return all valid arrangements (or count them).

* Word Search II: Given a grid of letters and a list of words, find all words that can be formed by traversing adjacent cells without reusing a cell in the same word.

* Number of Islands: Given a grid of land (1) and water (0), count how many separate islands exist. Adjacent land cells belong to the same island.

* Course Schedule: Given course prerequisites, determine whether it is possible to complete all courses. A cycle in the dependency graph makes it impossible.

* Clone Graph: Given a node in a connected graph, create a deep copy of the entire graph while preserving all connections.

* Lowest Common Ancestor of a Binary Tree: Given two nodes in a binary tree, find the lowest node that is an ancestor of both.

* Binary Tree Maximum Path Sum: Find the maximum possible sum of values along any path in a binary tree. The path can start and end anywhere but must follow parent-child connections.

* Longest Increasing Subsequence: Given an array of numbers, find the length of the longest subsequence whose values are strictly increasing.

* Coin Change: Given coin denominations and a target amount, determine the minimum number of coins needed to make that amount, or report that it's impossible.

* Merge k Sorted Lists: Merge k already-sorted linked lists into one sorted linked list efficiently.

* Trapping Rain Water: Given an elevation map represented as bars of different heights, calculate how much water would be trapped after rainfall.

* Sliding Window Maximum: Given an array and a window size k, return the maximum value in every contiguous window of size k as the window moves across the array.

* Find Median from Data Stream: Continuously receive numbers and efficiently return the median after each insertion.

* LRU Cache: Design a cache with a fixed capacity that supports O(1) lookups and updates while evicting the least recently used item when full.

* Word Ladder: Given a start word, an end word, and a dictionary, find the shortest sequence of valid one-letter transformations that converts the start word into the end word.
