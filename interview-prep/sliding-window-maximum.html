Sliding Window Maximum

Deque Hard

Given an array and a window size k, return the maximum value in every contiguous window of size k as the window moves across the array.

Approach (Monotonic Deque)

Maintain a deque of indices such that their corresponding values are in decreasing order. The front of the deque is always the maximum of the current window. Remove elements that fall outside the window and elements smaller than the new element from the back.

  • Time: O(N) — each element is inserted and removed at most once.
  • Space: O(k) for the deque.
public class Solution {
    public int[] MaxSlidingWindow(int[] nums, int k) {
        var deque = new LinkedList<int>();
        var result = new List<int>();

        for (int i = 0; i < nums.Length; i++) {
            // Remove indices out of this window
            if (deque.Count > 0 && deque.First.Value <= i - k)
                deque.RemoveFirst();

            // Remove smaller values from the back
            while (deque.Count > 0 && nums[deque.Last.Value] <= nums[i])
                deque.RemoveLast();

            deque.AddLast(i);

            // Start adding to result once window is full
            if (i >= k - 1)
                result.Add(nums[deque.First.Value]);
        }

        return result.ToArray();
    }
}
